# 裏技(Ad tips) - 広告素材取得方法

> **注意**: AdBlockにおける広告非表示が問題となっています。

このドキュメントは、各アダルトサイトから広告素材（画像・動画埋め込みコード）を取得する方法をまとめたものです。

---

## カリビアンコム

### 画像ZIP取得方法

画像を取得したい作品の詳細ページURLが下記の場合:

```
https://www.caribbeancompr.com/moviepages/113019_002/index.html
```

赤文字部分を変更すると、zipファイルパスになります。
（但し、作品によっては、提供されない場合もあります）:

```
https://www.caribbeancompr.com/moviepages/113019_002/images/gallery.zip
```

**URLパターン**: `index.html` → `images/gallery.zip`

### 動画埋め込みコード取得方法

動画詳細ページの「埋め込みプレーヤー」から、コードを取得しご利用ください。

**注意**: 「埋め込みプレーヤー」の表記がない場合は、サンプル動画が提供されない作品です。予め、ご了承ください。
